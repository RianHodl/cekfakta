# CekFakta.id

An AI-powered hoax detection web app designed to help Indonesian users verify the credibility of news and information circulating online.

Built for the **JuaraVibeCoding** competition by Google Developer Groups (May 2026) — recognized as **Top 100** out of all submissions.

---

## Features

- Input a news link or paste text directly
- AI analyzes the content and returns a hoax/valid verdict
- Includes a clear explanation of the actual facts
- Results are easy to read and share

## Tech Stack

- Google Gemini API
- Google AI Studio
- Google Antigravity
- HTML, CSS
